# Webcam-Image-Recognizer

A realtime system capable of seamlessly identifying thousands of common objects using a laptop's webcam feed, and then playing audio describing what the object is.

<h3> Requirements </h3>

- Python 3.7.3

- OpenCV2 4.1.0

- Tensorflow 1.13.1

- NumPy 1.16.4

- GTTS 2.0.3

- PyGame 1.9.6
